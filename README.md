## 📖 简介

**叮咚日历** 是一款“本地优先”、注重隐私的 Android 日程管理应用。它支持强大的重复规则，帮您轻松记录生日、纪念日、账单还款等生活琐事。

## 🚀 功能亮点

*   **强大的重复规则**：支持天、周、月、年及自定义规则（如“每月最后一个周五”、“每两周”）。
*   **完美支持农历**：深度集成农历算法，轻松记录农历生日、传统节日。
*   **隐私优先**：所有数据仅存储在本地。零追踪，无广告。
*   **安全备份**：支持将数据加密备份到您自己的 WebDAV 网盘（如坚果云、Nextcloud）。
*   **桌面组件**：美观实用、支持透明度调节的桌面日程小部件。
*   **经期助手**：内置私密的经期记录与预测功能。
